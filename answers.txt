1. Какими способами можно объявлять массивы в JS? 

    Пустые массивы:
        let arr1 = [];
        let arr2 = new Array(); 

    Массивы со значениями:
        let arr3 = ["one", 2, "3"] - массив из 3-х элементов: arr3[0] = "one"; arr3[1] = 2; arr[2] = "3";
        let arr4 = new Array(3) - массив также из 3-х элементов, но они пустые (undefined);

4. Можно ли пропускать части **for**? Что получится, если написать `for(;;)`? 

    Да, можно. Начало можно пропускать, если в начале цикла нам делать ничего не требуется, или нужная переменная уже объявлена;
    Если пропустить условие - цикл будет бесконечный;
    Если пропустить шаг - цикл будет подобен while(условие) (переменную для начала цикла следует объявлять до цикла);
    Если написать `for(;;)` - цикл будет бесконечный.

7. Как вывести чётные числа от 2 до 10 при помощи цикла for?

    for(let i = 1; i <= 10; i++) {
        if (!(i % 2)) {

        }

8. Каков будет результат выполнения этого кода? Почему?

    ```jsx
    let arr = ["a", "b"];

    arr.push(function() {
      alert( this );
    })

    arr[2](); // ?
    ```

    Результат: вывод в alert массива. Тоже самое было бы, если бы мы написали arr вместо ключевого слова this. То есть мы имеем в массиве функцию, которая выводит нам весь массив, обращаясь к нему через ключевое слово this.

11. Самостоятельно разберитесь, как можно проще всего сделать сортировку в массиве на JS? 

    Метод filter.
    
    let arr = [1, "2", "three", 4, 5, 6, true, "seven", false, 8];
    let numArr = arr.filter(item => typeof item === 'number' && !isNaN(item));
    console.log(numArr);
