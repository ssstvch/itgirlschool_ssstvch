1) Flexbox позволяется делать дизайн гибким и адаптивным, прост в применении
2) display: table; свойство полезно для динамического вертикальногои горизонтального выравнивания. Также позволяет сделать "прилипающий" футер, когда контента страницы мало, но скрывать его, когда контента становится больше. Плюс до сих пор не все браузеры (устаревшие) поддерживают flexbox.
3)  margin: 0  auto;
    text-align: center;
    . {
        display: flex;
        justify-content: center;
    }
4) Во flex-вёрстке есть две оси - главная и поперечная. Задать направление осей можно с помощью свойства flex-direction. 
5) margin:auto в горизонтальном выравнивании запоняет равномерно всё пространство, а в вертикальном 0 пикселей. Используется для выравнивания по горизонтали. 
6) box-sizing позволяет сохранять нужный размер элемента, включая в него padding и border, упрощая нам расчеты при написании стилей
7) flex-grow отвечает за количество занимаемого места flex-елемента относительно других flex-элементов, то есть он увеличивается, расширяется. а flex-shink отвечает за степень сжатия одного flex-элемента относительно других flex-элементов. (в голове понятно, а словами выразить не просто)
8) Позиционирование похоже на:
. {
    display: flex;
    justify content: space-between;
}
9) 142px x 16px
10) order отвечает за порядок flex-элементов. значение order не является порядковым номером элемента, а отвечает за его позиционирование относительно заданного. Отрицательные значение двигают элемент влево, положительные вправо.
