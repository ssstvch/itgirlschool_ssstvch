1. Что такое props и можно ли использовать props в функциональных компонентах?

    props это изменяемые значения компонента React для создания DOM-элементов. в функциональных компонентах они передаются в качестве аргумента, тогда как в классовых они объявляются как отдельная константа.

2. Нужно ли выделять в отдельный компонент статью в блоге?

    да, ведь статьи в блоге повторяются и меняется в них только контент.

3. Можно ли использовать React без JSX?

    да, можно. но придётся придётся писать ооочень длинный код с огромной вложенностью.

4. Чем отличается JSX от HTML?

    вместо class => className, а одиночные теги закрываются => <br />

5. Для чего нам нужны свойства (props) компонентов? 

    для создания компонентов, у которых мы можем менять содежимое DOM-элементов. это удобно, вместо дублирования одного и того же кода мы меняем лишь props.

11. Какой командой можно сгенерировать разметку/компоненты на основе заранее заданного массива элементов? Приведите пример.

    через props[index].prop, либо через props.map(prop)

    Пример:


    в файле компонента: 

    import "./Example.css"

    function Example(props) {
        <div className="wrapper">
            <h2 className="wraper__title">{props.title}</h2>
            <p className="wrapper__text">{props.text}<p>
        </div>
    }


    в файле приложения:

    import React from ('./React.js');
    import './App.css';
    import './components/Example.css';
    import Example from './components/Example'

    const array = [
        {
            title: 'Title 1',
            text: 'Text 1',
        },
        {
            title: 'Title 2',
            text: 'Text 2',
        },
        {
            title: 'Title 3',
            text: 'Text 3',
        },
    ];

    function Cat extends React.Component {
        const {title, text} = props;
        return (
            <React.Fragment>
                array.map((arr) =>
                    <Example title="title" text="text" />
                    <Example title="title" text="text" />
                    <Example title="title" text="text" />
                )
            </React.Fragment>
        )
    }
